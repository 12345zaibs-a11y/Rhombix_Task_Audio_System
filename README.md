# Rhombix Technologies Internship - Task 

## Project Overview
This project was completed as part of the Rhombix Technologies Unity Game Development Internship.

The objective of this task was to enhance a previously developed Unity game by implementing a complete audio system to improve the gameplay experience.

## Features
- 🎵 Background Music
- 🦘 Jump Sound Effect
- 🪙 Coin Pickup Sound
- ❤️ Player Hurt Sound
- 🤖 Enemy AI
- ❤️ Player Health System
- 💀 Enemy Health System
- 🪙 Coin Spawn on Enemy Death
- 🎮 2D Platformer Gameplay

## Tools Used
- Unity 6
- C#
- Visual Studio
